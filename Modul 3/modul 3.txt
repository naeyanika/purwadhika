## Modul 3 - boolean (Tipe Data Logika)
Tipe data boolean adalah tipe data yang hanya memiliki dua nilai, yaitu True dan False. Tipe data ini sering digunakan dalam pengkondisian dan perulangan.

Contoh penggunaan tipe data boolean:

```python
# Menggunakan boolean dalam pengkondisian
x = 5
y = 10
is_greater = x > y
print(is_greater)  # Output: False

# Menggunakan boolean dalam perulangan
while is_greater:
    print("x lebih besar dari y")
